# Weather Forecasting using Machine Learning

This project predicts temperature using historical weather data and machine learning techniques.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Jupyter Notebook

## Dataset
The dataset contains weather parameters such as temperature, humidity, wind speed, cloud cover, pressure, and rainfall.

## Model Used
- Linear Regression

## Steps
1. Data loading
2. Data cleaning
3. Feature selection
4. Model training
5. Evaluation

## Result
The model predicts temperature with acceptable accuracy using Mean Absolute Error.
